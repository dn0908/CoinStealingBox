## Hardware Specifications
- PyMODI kit : LED, Button, Dial, Speaker, Display, SerialCommunication Module used
- Arduino UNO
- L9110 / HG7881 2 Channel Motor Driver
- TT Motor DC Gearbox Motor Dual Shaft 200RPM DC 3-6V
- VISVI STUDY9 Webcam
